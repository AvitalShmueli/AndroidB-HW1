# Android B - HW1 - multi-app project

Home Assignment Exercise for Android B course, in Afeka - the academic college
of Engineering in Tel Aviv.

## Overview

This is a multi-app project that includes 2 apps with a common library.
One app is a water tracker and the other is caffeine tracker.
Each app uses shared preferences and has an independent data manager.

- **Frontend:** Java

## Water Tracker

The user clicks on the "Add amount" button and chooses the amount option from the list.
When clicking on the save button, the amount is added to today's water amount.
The user's goal is to reach 2 liters per day and keeping a streak of reaching the goal.

## Caffeine Tracker

The user clicks on the "Add amount" button and chooses the beverage option from the list - each beverage contains a different amount of caffeine.
When clicking on the save button, the caffeine amount is added to today's amount.
The user's goal is to not consume more than 400 mg of caffeine per day and keeping a streak of reaching the goal.
